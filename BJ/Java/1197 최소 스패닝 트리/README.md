## 1197 최소 스패닝 트리

https://www.acmicpc.net/problem/1197

---

- 크루스칼 알고리즘
    > 간선의 가중치의 합이 최솟값이 되도록 모든 노드를 연결
    1. 간선 데이터를 오름차순으로 정렬
    2. 간선을 확인하며 현 간선이 사이클을 발생시키는지 확인
        - 발생시키지 않으면 MST에 포함
    3. 모든 간선에 대해 반복
- 프림 알고리즘
    > 시작 정점을 기준으로 가장 작은 간선과 연결된 정점을 선택하여 모든 노드를 연결
    1. 임의의 간선 선택
    2. 현 정점으로부터 가장 낮은 가중치를 갖는 정점 선택
    3. 모든 간선에 대해 반복
